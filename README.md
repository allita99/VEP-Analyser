# VEP-Analyser-Motherboard

This repository contains an updated version of the hardware and software developed during my Bachelor's thesis, with incremental improvements and ongoing refinements.

The project integrates multiple components required for a VEP-based system, including firmware, sensor hardware, and the main motherboard design.

---

## Contents

* **Microcontroller Code**
  Firmware used for signal acquisition, control, and system interaction.

* **Motherboard Design**
  Central hardware platform responsible for system integration, power distribution, and communication between components.

* **VEP Sensor (VEPsor) Design**
  Hardware for acquiring visual evoked potential signals.
